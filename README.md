# Game of Life

Simple implementation of "life game"
https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life


## setup
```
cp .env_sample .env
```
optional: edit and update .env

## run

### with docker
```
docker-compose up
```

### local node / npm:
```
npm install
npm run dev
```

open browser http://localhost:3003
